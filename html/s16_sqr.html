NAME

s16_sqr - optimized for x^2  

SYNOPSIS

#include "s16math.h"
qm_n s16_sqr(qm_n x);
 

DESCRIPTION

Evaluate x^2 and return result.  

PARAMETER

x
The parameter is supplied as a 16 bit integer having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution.
 

RETURN

The return value is a 16 bit integer having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution.

A negative return value means overflow has occurred.  

NOTES

Internal 32 bit intermediate results.

This operation is implemented as a macro.
