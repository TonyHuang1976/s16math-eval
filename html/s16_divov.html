NAME

s16_divov - Check for divide overflow  

SYNOPSIS

#include "s16math.h" s16 s16_divov(qm_n a, qm_n b, qm_n c);
 

DESCRIPTION

Divide or multiply overflow occurs when the result is positive and the operands are of different signs, or if the result is negative and the operands are of the same sign. Otherwise there is no overflow.

In the case where n != 0, n/0 is always deemed to be overflow and 0/n is never deemed to be overflow.  

PARAMETERS

a
quotient.
b
dividend.
c
divisor.
 

RETURN

Zero if no overflow. Non-zero if overflow.  

NOTE

A value of zero for the quotient when the dividend and divisor are both non- zero is not by itself considered an error, since the condition may occur due to rounding.
