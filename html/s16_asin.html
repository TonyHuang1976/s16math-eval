NAME

s16_asin - arc sine  

SYNOPSIS

#include "s16math.h" qm_rad s16_asin(qm_n x);
 

DESCRIPTION

Computes arc sine from sine. Return value saturates for sine values >= 100 - i.e. 1.00.  

PARAMETER

The sine parameter is supplied as a 16 bit integer having an implied decimal point with two digits of resolution, or an implied binary point with seven bits of resolution.
x
numeric argument.
 

RETURN

The return value is the 16 bit arc sine in radians represented as an integer having an implied decimal point with three digits of resolution, or an implied binary point with 10 bits of resolution.
