NAME

s16_cot - cotangent  

SYNOPSIS

#include "s16math.h" qm_n s16_cot(qm_rad phi);
 

DESCRIPTION

Computes 1/tan(phi) by evaluating cos/sin.  

PARAMETER

Parameter is radians supplied as a 16 bit integer having an implied binary point with 10 bits of resolution, or an implied decimal point with three digits of resolution.
phi
numeric argument in radians.
 

RETURN

The return value is a 16 bit integer cotangent having an implied binary point with seven bits of resolution, or an implied decimal point with two digits of resolution.
y
1/tan(phi). For large values of the trigonometric ratio, the return value saturates at positive or negative infinity.

Positive and negative infinity are indicated by return values of 327.67 and -327.68 respectively when compiled with decimal resolution, or 255.127 and -256.000 when compiled with binary resolution.  

NOTES

Internal 32 bit intermediate results.

The accuracy of this computation is poor. For angles between -45 degrees and +45 degrees inclusive, only 82% of results correspond to those of the reference implementation to within +/- 0.1 units, and only 38% correspond to within +/- 0.01 units. Correspondence declines further with increasing angle.
