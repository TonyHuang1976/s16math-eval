NAME

s16_cos - cosine  

SYNOPSIS

#include "s16math.h"
qm_n s16_cos(qm_rad phi);
 

DESCRIPTION

Computes cosine from angle.  

PARAMETER

phi
is an angle in radians represented as an integer having an implied binary point with 9 bits of resolution, or an implied decimal point with three digits of resolution.
 

RETURN

The return value is the 16 bit cosine represented as an integer having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution. Range is -1.00 through 1.00.
