NAME

s16_sqrt - evaluate square root  

SYNOPSIS

#include "s16math.h" qm_n s16_sqrt(qm_n a);
 

DESCRIPTION

Evaluate a^(1/2) and return principal root.  

PARAMETER

The parameter is supplied as 16 a bit integer having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution.
a
number to find square root of.
 

RETURN

The root is returned as a 16 bit integer having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution.

If the square root of a negative number is requested, the function returns zero.  

NOTES

Internal 32 bit intermediate results.
