NAME

s16_addov - Check for add overflow  

SYNOPSIS

#include "s16math.h" s16 s16_addov(qm_n a, qm_n b, qm_n c);
 

DESCRIPTION

Add overflow occurs when the operands have the same sign and the sum has a different sign. Otherwise there is no overflow.  

PARAMETERS

a
sum.
b
augend.
c
addend.
 

RETURN

Zero if no overflow. Non-zero if overflow.
