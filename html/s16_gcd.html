NAME

s16_gcd - greatest common divisor  

SYNOPSIS

#include "s16math.h"
u16 s16_gcd(u16 u, u16 v);
 

DESCRIPTION

Computes greatest common divisor of two numbers. The greatest common divisor is useful for reducing fractions to be in lowest terms. For example, s16_gcd(42, 56) = 14. Therefore, both parameters can be divided by 14 to yield 42/14 = 3, and 56/14 = 4.  

PARAMETERS

Both parameters are unsigned 16 bit integers.  

RETURN

The return value is the signed 16 bit greatest common divisor. Range is -32768 through 32767.
