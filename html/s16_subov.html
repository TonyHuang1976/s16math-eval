NAME

s16_subov - Check for subtract overflow  

SYNOPSIS

#include "s16math.h" s16 s16_subov(qm_n a, qm_n b, qm_n c);
 

DESCRIPTION

Subtract overflow occurs when the operands have different signs and the remainder has the same sign as the subrahend. Otherwise there is no overflow.  

PARAMETERS

a
remainder.
b
minuend.
c
subtrahend.
 

RETURN

Zero if no overflow. Non-zero if overflow.
