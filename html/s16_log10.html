NAME

s16_log10 - common logarithm  

SYNOPSIS

#include "s16math.h"
qm_n s16_log10(qm_n x);
 

DESCRIPTION

Evaluates log to base 10 of x, where 0 < x <= 327.67. The logarithm is evaluated as y=s16_ln(x)/ln(10). A constant is used for ln(10). So requesting the log of a negative number is treated as an error instead of a multiple of i.  

PARAMETERS

x
numeric argument with equivalent of two decimal digits of resolution.
 

RETURN

If 0 < x <= 327.67, then the return value is the number to which 10 would have to be raised to yield x. A return of -99.00 means x <= 0 or x > 327.67.
