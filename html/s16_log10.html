NAME

s16_log10 - common logarithm  

SYNOPSIS

#include "s16math.h" qm_n s16_log10(qm_n a);
 

DESCRIPTION

Evaluates log to base 10 of a, where 0 < a <= 327.67 (decimal resolution), or 255.127 (binary resolution). The logarithm is evaluated as s16_ln(a)/ln(10). A constant is used for ln(10).  

PARAMETERS

a
numeric argument with equivalent of two decimal digits or seven binary bits of resolution.
 

RETURN

If 0 < a <= 327.67, then the return value is the number to which 10 would have to be raised to yield a.

A return of -99.00 (S16_NAN) means a <= 0. By convention, a = 327.68 (256.000) is taken to be -327.68 (-256.000) and results in an error return.
