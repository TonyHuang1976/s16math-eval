NAME

s16_log - log to aribitrary base  

SYNOPSIS

#include "s16math.h" qm_n s16_log(qm_n a, qm_n n);
 

DESCRIPTION

Evaluates log to base a of n, where 0 < n <= 327.67 (decimal resolution), or 255.127 (Q8.7).

The logarithm is evaluated as s16_ln(n)/s16_ln(a).  

PARAMETERS

Parameters are supplied as 16 bit integers having an implied decimal point with two digits of resolution or an implied binary point with seven bits of resolution.
a
base of logarithm.
n
numeric argument with two decimal digits or seven bits of resolution.
 

RETURN

The return value is a 16 bit integer having an implied decimal point with two digits of resolution or an implied binary point with seven bits of resolution.

If 0 < n <= 327.67 (255.127 if binary resolution), then the return value is the number to which base a would have to be raised to yield n.

A return of -99.00 (S16_NAN) means a <= 0. By convention, a = 327.68 (256.000) is taken to be -327.68 (-256.000) and results in an error return.  

NOTES

The accuracy of this function is very poor. Due to limitations in precision and rounding errors most results differ from those of the reference implementation by more than one part in 100. Some results differ from those of the reference implementation by as much as seven parts in 100.
