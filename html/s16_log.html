NAME

s16_log - log to aribitrary base  

SYNOPSIS

#include "s16math.h"
qm_n s16_log(qm_n a, qm_n n);
 

DESCRIPTION

Evaluates log to base a of n, where 0 < a,n <= 327.67. So requesting the log of a negative number is treated as an error instead of a multiple of i.

a and n are entered as rational numbers with two digits to the right of a virtual decimal point. The logarithm is evaluated as y=s16_ln(n)/s16_ln(a).  

PARAMETERS

Parameters are supplied as 16 bit integers having an implied decimal point with two digits of resolution.
a
base of logarithm.
n
numeric argument with equivalent of two decimal digits of resolution.
 

RETURN

The return value is a 16 bit integer having an implied decimal point with two digits of resolution.

If 0 < a,n <= 327.67, then the return value is the number to which base a would have to be raised to yield n. A return of -99.00 means n <= 0 or n > 327.67.
