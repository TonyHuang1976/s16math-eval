NAME

s16_csc - cosecant  

SYNOPSIS

#include "s16math.h" qm_n s16_csc(qm_rad phi);
 

DESCRIPTION

Computes 1/sin(phi).  

PARAMETER

Parameter is radians supplied as a 16 bit integer having an implied binary point with 10 bits of resolution, or an implied decimal point with three digits of resolution.
phi
numeric argument in radians.
 

RETURN

The return value is a 16 bit integer cosecant having an implied binary point with seven bits of resolution, or an implied decimal point with two digits of resolution.
y
1/sin(phi).

For large values of the trigonometric ratio, the return value saturates at the maximum possible positive or negative value for the resolution scheme in the regions around the zeroes of sin(phi) - 0 and 180 deg. For phi in the first or second quadrant, the saturation value is 327.67 (255.127). For phi in the third or fourth quadrant, the saturation value is -327.68 (-256).  

NOTES

Internal 32 bit intermediate results.

The accuracy of this computation is poor. For angles between -45 degrees and +45 degrees inclusive, only 82% of results correspond to those of the reference implementation to within +/- 0.1 units, and only 38% correspond to within +/- 0.01 units. Correspondence declines further with increasing angle.
