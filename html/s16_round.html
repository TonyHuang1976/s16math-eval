NAME

s16_round - round to nearest, ties to even  

SYNOPSIS

#include "s16math.h" qm_n s16_round(qm_n x);
 

DESCRIPTION

Rounds to the nearest real value [1]; if the number falls midway, it is rounded to the nearest real value with an even next significant digit; this corresponds to the IEEE default. Where the least significant digit is zero, the number is unchanged.

Where rounding up would result in overflow (change of sign), the number is truncated (rounded toward zero) instead.  

PARAMETER

x
Parameter is supplied as a 16 bit integer having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution.
 

RETURN

Where there would be no overflow, the return value is a 16 bit real valued number rounded up or down to the nearest even next significant digit. Otherwise the return value is the truncated number, or is the unchanged number.  

NOTES

Internal use.
