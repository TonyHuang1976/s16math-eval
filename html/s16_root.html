NAME

s16_root - evaluate root  

SYNOPSIS

#include "s16math.h"
qm_n s16_root(qm_n a, qm_n n);
 

DESCRIPTION

Evaluate a^(1/n) and return principal root.  

PARAMETERS

Parameters are supplied as 16 bit integers having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution.
a
number to find root of.
n
root to be found.
 

RETURN

The root is returned as a 16 bit integer having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution.

If an even root of a negative number is requested, the function returns zero. Otherwise the function returns a negative root [8].

A negative return value of -327.68, or -216.000 if binary resolution is used, means overflow has occurred.  

NOTES

Internal 32 bit intermediate results.

Calculators vary in handling roots of negative numbers. The funcs.bc package [8] issues an error message followed by a zero when an even integer valued root of a negative number is requested. The Debian qalculate-gtk desktop calculator package returns a complex number. This function follows funcs.bc practice.

For odd integer valued roots of negative numbers, both packages return the negative of the root of the corresponding positive number, as does this function.
