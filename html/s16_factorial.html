NAME

s16_factorial - n!  

SYNOPSIS

#include "s16math.h"
qm_n s16_factorial(qm_n n);
 

DESCRIPTION

Evaluates n!.  

PARAMETERS

n
numeric argument in 16 bit fixed point decimal with two digits of resolution.
 

RETURN

A negative return indicates overflow. Otherwise it is the factorial of n.  

NOTE

Attempting to evaluate the factorial of numbers greater than 5.5 (550 in the internal representation) produces overflow.  

NOTES

Internal 32 bit intermediate results.
