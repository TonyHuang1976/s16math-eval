NAME

s16_factorial - n!  

SYNOPSIS

#include "s16math.h"
qm_n s16_factorial(qm_n n);
 

DESCRIPTION

Evaluates n!. Where n is negative, n! = 0. Where n is 0 or 1, n! = 1.  

PARAMETERS

The parameter is supplied as a 16 bit integer having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution.
n
numeric argument.
 

RETURN

Zero is returned for the factorial of a negative number. The factorial of 0 or 1 is defined to be 1. A negative return indicates overflow. Otherwise it is the factorial of n.  

NOTE

Attempting to evaluate the factorial of numbers greater than 5.5 (550 using decimal resolution), or greater than 5.3671 - 5 and 47/128 - with binary resolution, produces overflow.  

NOTES

Internal 32 bit intermediate results.
