Source License

The source license makes the full source for all the math functions, test code, and build facilities available to you. You may optimize the library code for your specific application and incorporate the result into your own product. You can build customized versions of the library using both "Make" variables that can be overridden on the command line, and by selecting among specified conditional compilation options available in the product source files.

The following enumerated source change types are supported:

First, variables defined in the file "Makefile" in the top level directory as: 

    Name     Assignment
    ----     ----------
    var      ?=[<initial value>]

may be overridden on the command line.

Second, conditional definition macros of the form: 

    #ifdef     USE_<filename><option>
    #undef     USE<filename>_<option>

where <filename> is an upper case version of the file name (without the ".c" extension) and <option> is a descriptive term such as "TABLE", or "NTHROOT", may be interchanged to allow you to select among algorithms where more than one algorithm is available.

Code within the scope of undefined USE_xxx constants compiles - mostly (I think) - but may not work. In most cases disabled algorithms may use less code, but provide poorer accuracy, than the one enabled by default.

Support for six months is included in the price, and is available after that for an hourly fee. Source updates are provided at no charge.

The source license is available for a fixed fee. Binaries made from either modified or unmodified source can be distributed for a royalty set as a percentage of the unit sale price.

If the source license is terminated, you no longer pay royalties, and you may no longer include the library in your product.

You may make any other source code changes you wish. Support for those changes is available for an hourly fee plus materials.

If accepted, the patch may be modified in any way. The resulting code will be made available as part of the product and provided with standard support.
