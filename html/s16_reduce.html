NAME

s16_reduce - reduction of terms  

SYNOPSIS

#include "s16math.h"
s16 s16_reduce(qm_n k);
 

DESCRIPTION

Computes reduction of terms for k to unit integer - either 100 (decimal resolution) or 128 (Q8.7 resolution). Evaluates fractional part of k for k/UI=x/y for x and y.  

PARAMETERS

k
16 bit signed fixed point number with two digits or seven bits of resolution.
 

RETURN

x and y are packed as two signed bytes into a 16 bit signed integer. x is returned in bit 15:8, and y is returned in bits 7:0. To be used in fixed point operations, both x and y must be scaled by the unit integer.

The function returns zero in bits 7:0 if k is zero or greater than the unit integer.
