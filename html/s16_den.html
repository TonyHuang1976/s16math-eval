NAME

s16_den - convert integer portion to denominator.  

DESCRIPTION

Converts the fractional part of a 16 bit signed fixed point number to a fixed point value.

The fractional portion of the parameter is treated as the numerator of a fraction with an implied denominator of 100. It is divided into 100 and the integer portion of the result is returned in the form n.00.  

PARAMETER

pwr
real number power with integer and fractional parts.
 

RETURN

The return value is the signed 16 bit denominator. Range is -327.00 through 327.00.  

NOTE

Superceded by s16_reduce.
