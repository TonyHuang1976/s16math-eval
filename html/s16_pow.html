NAME

s16_pow - evaluate power  

SYNOPSIS

#include "s16math.h"
qm_n s16_pow(qm_n a, qm_n n);
 

DESCRIPTION

Evaluate a^n and return power.

Fractional operands are evaluated by applying successive exponential identities to simplify them into a set of integer valued power and root operations of acceptable degree.

The resolution of fractional exponents is constrained by using a reduction table to obtain manageable values for power/root calculations. This approximates the fractional part of the exponent to the nearest 1/16'th value when binary resolution is used and the nearest 1/10'th value when decimal resolution is used.

Accuracy is good for bases with exponents that are integer multiples of 0.1 for decimal resolution and 1/16'th for binary resolution. Results for other fractional exponents can vary by up to +/- 0.15 from the mathematically correct value.  

PARAMETERS

Parameters are supplied as 16 bit integers having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution.
a
base to be exponentiated.
n
exponent.
 

RETURN

The power is returned as a 16 bit integer having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution.  

NOTES

Internal 32 bit intermediate results.
