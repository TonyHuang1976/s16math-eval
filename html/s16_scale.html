NAME

s16_scale - scale number by scale factor  

SYNOPSIS

#include "s16math.h"
s16 s16_scale(s16 n,s16 s);
 

DESCRIPTION

Reduces resolution of number by scaling factor.  

PARAMETERS

n
positive number expressed to decimal or Qm.n resolution.
s
positive scaling factor: shift factor if compiled with binary option, denominator if decimal compiled with decimal option.
 

RETURN

The return value is the scaled number.  

NOTES

Internal use.
