NAME

s16_sqrov - Check for squaring overflow  

SYNOPSIS

#include "s16math.h"
s16 s16_sqrov(qm_n a);
 

DESCRIPTION

Squaring overflow occurs when the result of squaring a number is negative or the number is negative. Otherwise there is no overflow.  

PARAMETER

a
The parameter is supplied as a 16 bit integer having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution.
 

RETURN

Zero if no overflow. Non-zero if overflow.  

NOTE

This operation is implemented as a macro.
