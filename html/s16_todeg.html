NAME

s16_todeg - radians to degrees  

SYNOPSIS

#include "s16math.h" qm_deg s16_todeg(qm_rad rad);
 

DESCRIPTION

Convert radians with three digits - or 10 bits - of resolution to degrees with one digit - or 4 bits - of resolution.  

PARAMETER

rad
is an angle in radians having an implied binary point with 10 bits of resolution, or an implied decimal point with three digits of resolution.
 

RETURN

deg
The return value is a 16 bit integer degree value having an implied binary point with 4 bits - or one decimal digit - of resolution.

If deg differs in sign from rad, overflow has occured.  

NOTES

Internal 32 bit intermediate results.
