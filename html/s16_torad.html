NAME

s16_torad - degrees to radians  

SYNOPSIS

#include "s16math.h"
qm_rad s16_torad(qm_deg deg);
 

DESCRIPTION

Convert degrees with one decimal digit - or 4 bits - of resolution to radians with three digits - or 9 bits - of resolution.  

PARAMETER

deg
is an angle in degrees represented as an integer having an implied binary point with 4 bits of resolution, or an implied decimal point with one digit of resolution.
 

RETURN

rad
The return value is a 16 bit integer radian value having an implied binary point with 9 bits of resolution, or an implied decimal point with three digits of resolution.

If rad differs in sign from deg, overflow has occured.  

NOTES

Internal 32 bit intermediate results.
