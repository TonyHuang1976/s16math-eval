NAME

s16_div - divide  

SYNOPSIS

#include "s16math.h" qm_n s16_div(qm_n a, qm_n b);
 

DESCRIPTION

Computes quotient of two numbers.  

PARAMETERS

a
the first parameter is the dividend supplied as a 16 bit integer having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution.
b
the second parameter is the divisor supplied as a 16 bit integer having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution.
 

RETURN

The return value is the 16 bit quotient represented as an integer having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution. Range is -327.68 through 327.67 (-256.000 through 255.127).

The function returns 0 when divide by zero is attempted.  

NOTES

Internal 32 bit intermediate results.
