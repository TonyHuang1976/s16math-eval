NAME

s16_ln - natural logarithm  

SYNOPSIS

#include "s16math.h" qm_n s16_ln(qm_n a);
 

DESCRIPTION

Evaluates log_e(a), where 0 < a <= 327.67 (decimal resolution), or 255.127 (Q8.7). Defined for any positive real number a as the area under the curve y = 1/x from 1 to a (the area being taken as negative when a < 1 [2]).  

PARAMETERS

a
The number.
 

RETURN

If 0 < a <= 255.127 (327.67), then the return value is the number to which e would have to be raised to yield a.

A return of -99.00 (S16_NAN) means a <= 0. By convention, a = 327.68 (256.000) is taken to be -327.68 (-256.000) and results in an error return.  

NOTES

Internal 32 bit intermediate results.

The function uses a series expansion algorithm based on accuracy and performance.

Better accuracy was obtained using direct Reimann integration. However, series expansion was found to run 15 times faster on the selected MCU for large values of tangent. Only 0.04% of series expansion results differed from those of the reference by more than 0.01, and no results differed from the reference values by more than 0.02.
