NAME

s16_ln - natural logarithm  

SYNOPSIS

#include "s16math.h"
qm_n s16_ln(qm_n n);
 

DESCRIPTION

Evaluates log_e(n), where 0 < n <= 327.67 (decimal resolution), or 255.127 (Q8.7). Requesting the log of a negative number is treated as an error instead of a multiple of i.  

PARAMETERS

n
The number.
 

RETURN

If 0 < n <= 255.127 (327.67), then the return value is the number to which e would have to be raised to yield n. A return of -99.00 means n <= 0 or n > 327.67 (255.127).  

NOTES

Internal 32 bit intermediate results.
