NAME

s16_logov - Check for natural log, log base 10, or log base n overflow.  

SYNOPSIS

#include "s16math.h" qm_n s16_logov(qm_n a);
 

DESCRIPTION

Checks a for equality with S16_NAN (not-a-number).  

PARAMETERS

a
value returned from log function
 

RETURN

Zero if no overflow. Non-zero if overflow.  

NOTES

Implemented as a macro.

s16_ln(3m), s16_log(3m), s16_log10(3m)
