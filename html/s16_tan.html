NAME

s16_tan - tangent  

SYNOPSIS

#include "s16math.h" qm_n s16_tan(qm_rad phi);
 

DESCRIPTION

Computes tangent from angle.  

PARAMETER

phi
is an angle in radians represented as an integer having an implied binary point with 10 bits of resolution, or an implied decimal point with three digits of resolution.
 

RETURN

The return value is the 16 bit tangent represented as an integer having an implied binary point with seven bits of resolution, or an implied decimal point with two digits of resolution.  

NOTE

Accuracy for angles from 0 to 51.9 degrees corresponds with the that of the reference implementation to within +/- 2 percent. Accuracy declines at angles between 1.571 +|- 0.67 radians (90 +|- 38 degrees) and 4.712 +|- 0.67 radians (270 +|- 38 degrees).

Positive and negative infinity are indicated by return values of 327.67 and -327.68 respectively when compiled with decimal resolution, or 255.127 and -256.000 when compiled with binary resolution.
