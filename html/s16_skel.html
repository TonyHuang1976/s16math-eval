NAME

s16_skel - skeleton function for software fixed point math library.  

SYNOPSIS

#include "s16math.h"
qm_n s16_skel(qm_n x);
 

DESCRIPTION

Returns its parameter.  

PARAMETERS

x
Parameter is supplied as a 16 bit integer having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution.
 

RETURN

The return value is a 16 bit integer having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution.  

NOTES

To italicize, say, y to y, use "\fIy\fP" (without the quotes).
