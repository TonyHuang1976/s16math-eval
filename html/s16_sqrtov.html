NAME

s16_sqrtov - Check for square root overflow.  

SYNOPSIS

#include "s16math.h" qm_n s16_sqrtov(qm_n a, qm_n b);
 

DESCRIPTION

Overflow is deemed to have occurred if the root is zero or negative. Otherwise, there is no overflow.  

PARAMETERS

a
root
b
radicand

Parameters are supplied as 16 bit integers having implied binary points with 7 bits of resolution, or implied decimal points with two digits of resolution.  

RETURN

Zero if no overflow. Non-zero if overflow.  

NOTES

This function is implemented as a macro. Probably does not catch all cases.
