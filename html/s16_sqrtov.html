NAME

s16_sqrtov - Check for square root overflow.  

SYNOPSIS

#include "s16math.h"
qm_n s16_sqrtov(qm_n a);
 

DESCRIPTION

Overflow is deemed to have occurred if the root is zero or negative. Otherwise, there is no overflow.  

PARAMETERS

a
root

The parameter is supplied as a 16 bit integer having an implied binary point with 7 bits of resolution, or an implied decimal point with two digits of resolution.  

RETURN

Zero if no overflow. Non-zero if overflow.  

NOTE

This function is implemented as a macro.
