NAME

s16_hypot - hypoteneuse  

SYNOPSIS

#include "s16math.h"
qm_n s16_hypot(qm_n x, qm_n y);
 

DESCRIPTION

Computes hypoteneuse from adjacent and opposite sides of a right triangle.  

PARAMETERS

x, y
16 bit integers having implied decimal points with two digits of resolution.
 

RETURN

The return value is the signed 16 bit hypoteneuse. Range is -327.68 through 327.67.
