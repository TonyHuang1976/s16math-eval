NAME

s16_acos - arc cosine  

SYNOPSIS

#include "s16math.h" qm_rad s16_acos(qm_n x);
 

DESCRIPTION

Computes arc cosine from cosine. Return value saturates for cosine values >= 100 - i.e. 1.00.  

PARAMETER

The cosine parameter is supplied as a 16 bit integer having an implied decimal point with two digits of resolution, or an implied binary point with seven bits of resolution.
x
numeric argument.
 

RETURN

The return value is the 16 bit arc cosine in radians represented as an integer having an implied decimal point with three digits of resolution, or an implied binary point with 10 bits of resolution.
